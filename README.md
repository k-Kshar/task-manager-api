# Task Manager API (FastAPI)

## Особенности
REST API для управления задачами.  
Реализован на FastAPI с использованием Pydantic и in-memory хранения данных.

## Описание
Это backend-приложение для управления задачами.

Реализованы:

- создание задач
- получение списка задач
- получение задачи по ID
- обновление задач
- удаление задач

## Технологии
- Python
- FastAPI
- Pydantic
- REST API архитектура
- Swagger UI

## Как запустить

1. Установить зависимости:

pip install fastapi uvicorn


2. Запустить сервер:

uvicorn app.main:app --reload


3. Открыть в браузере:

http://127.0.0.1:8000/docs


## Пример функционала

Пользователь может:
- создавать задачи
- просматривать список задач
- обновлять задачи
- удалять задачи
