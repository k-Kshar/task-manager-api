# Task Manager API (FastAPI)

## Особенности
Task Manager API — это backend-приложение для управления задачами, разработанное на FastAPI.

Проект реализует базовую CRUD-логику и демонстрирует навыки построения REST API, работы с валидацией данных и архитектурой backend-приложения.

## Описание
Это backend-приложение для управления задачами.

Реализованы:

- создание задач
- получение списка задач
- получение задачи по ID
- обновление задач
- удаление задач

## 🧠 Особенности реализации

- REST API архитектура
- FastAPI framework
- Pydantic для валидации данных
- Разделение логики через router (api слой)
- Swagger документация (/docs)
- In-memory хранение данных (без БД на текущем этапе)

## Технологии
- Python 3.10+
- FastAPI
- Pydantic
- REST API архитектура
- Swagger UI

## Как запустить

1. Установить зависимости:

pip install fastapi uvicorn


2. Запустить сервер:

uvicorn app.main:app --reload


3. Открыть в браузере:

http://127.0.0.1:8000/docs


## Пример функционала

Пользователь может:
